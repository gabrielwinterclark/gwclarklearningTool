Learning Objective 2: Understand intrinsic vs. extrinsic value

  1. Skill 1: Identify when someone is assigning intrinsic value to something (second definition)
  2. Skill 2: Identify when someone is assigning extrinsic value to something

In the last learning objective, we learned about the first definition of intrinsic value. Now, we will move on to the second definition of intrinsic value, and its corresponding “opposite”: extrinsic value.

Skill 1: Identify when someone is assigning intrinsic value to something (second definition)

By the second definition, a thing has intrinsic value when its value arises from nonrelational properties, or properties that are separate from that thing.

For example, I think the painting Starry Night by Vincent Van Gogh is valuable because of its colors and shapes. I don’t value it because it was painted by a famous artist, or because it makes me feel something. I value the painting for its nonrelational properties. The colors and shapes are inseparable from the painting. So, I value it intrinsically by this second definition.

Imagine the painting in a vacuum. No, not that kind of vacuum! I’m talking about a sort of void, consisting of nothing else but the painting. One way to test if something has intrinsic value by this second definition is to imagine it in a vacuum. Does the painting have value all by itself in the vacuum? If so, that value probably comes from its nonrelational properties - those things that can’t be separated from it. And, if this is true, that thing has intrinsic value.

Skill 2: Identify when someone is assigning extrinsic value to something

In the last Learn by Doing activity, we saw that Micah values Ratatouille because it makes them and their friend laugh. The fact that Ratatouille makes Micah laugh is a relational property of the movie: it can be separated from the movie. And, because this is what makes Micah value the movie, we say that it is extrinsically valuable. Where valuing something intrinsically by the second definition means its value comes from its nonrelational (inseparable) properties, valuing something extrinsically means its value comes from its relational (separable) properties.
