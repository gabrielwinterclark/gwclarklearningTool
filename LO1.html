Learning Objective 1: Understand intrinsic vs. instrumental value

Skills:

  1. Skill 1: Identify when someone is assigning intrinsic value to something (first definition)
  2. Skill 2: Identify when someone is assigning instrumental value to something

In philosophy, there are actually two definitions of intrinsic value. Each definition has a corresponding “opposite” sort of value. In this learning objective, we will learn about the first definition of intrinsic value and how it contrasts with its “opposite”: instrumental value. We will learn about the second definition in the next learning objective.

Skill 1: Identify when someone is assigning intrinsic value to something (first definition)

By the first definition, something has intrinsic value when it is valuable as an end. What does “valuable as an end” mean? Well, something is valuable as an end when it’s valuable for itself, not in service of something else.

Let's see if an example can help us understand even better.

Example: I love sushi because it’s delicious, not because it gets or does anything for me. The sushi doesn’t grow legs, buy a car and start running all my errands for me. No, I love sushi just because I enjoy eating it. I value sushi intrinsically. For me, the sushi has intrinsic value.

Try thinking of something that you find intrinsically valuable. What do you find interesting, enjoyable, or important on its own?

Skill 2: Identify when someone is assigning instrumental value to something

By the first definition, something has intrinsic value when it’s valued as an end, for itself. In contrast, something has instrumental value when it’s valued because as a means to get something else. In the previous Learn By Doing exercise, you heard about May, who values playing the cello instrumentally. She values playing the cello because it pays her bills: for May, playing the cello is a means to get money.
